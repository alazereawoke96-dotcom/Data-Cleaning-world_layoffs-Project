🧹 World Layoffs Data Cleaning Project (SQL)

📌 Project Overview
This project focuses on cleaning and preparing a global layoffs dataset using SQL. Raw datasets often contain duplicates, and missing values, which can affect analysis accuracy.  The goal of this project is to transform the raw layoffs data into a clean, reliable, and analysis-ready dataset using SQL data cleaning techniques.

🎯 Objectives
Remove duplicate records
Standardize inconsistent data
Handle missing and null values
Convert data types where necessary
Prepare the dataset for exploratory data analysis

📂 Dataset
The dataset contains global layoffs information including:
Company name
Location
Industry
Total layoffs
Percentage of layoffs
Date of layoffs
Funding stage
